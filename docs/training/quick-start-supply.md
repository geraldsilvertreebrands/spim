# Supply Panel Quick Start Guide

**Get insights in 5 minutes**

---

## 1. Login

- Go to your Silvertree platform URL
- Enter your supplier credentials
- You'll be redirected to `/supply`

---

## 2. View Dashboard KPIs

The dashboard shows:

- **Total Sales**: Revenue for selected period
- **Units Sold**: Quantity sold
- **Average Order Value**: Avg basket size
- **Unique Customers**: Number of buyers
- **Gross Margin**: Profit margin %
- **Stock Coverage**: Days of inventory

**Change period**: Use dropdown (Last 7/30/90 days, YTD, etc.)

---

## 3. Analyze Product Performance

**Products** page:

1. Click **Products** in navigation
2. See sales data per product
3. Sort by revenue, units, or stock
4. Search for specific products

---

## 4. View Sales Trends

**Trends** page:

1. Click **Trends** in navigation
2. View sales over time (daily/weekly/monthly)
3. Compare to previous period or last year
4. Identify seasonal patterns

---

## 5. Check Benchmarks

**Benchmarks** page:

1. Click **Benchmarks** in navigation
2. See how you compare to category average
3. View:
   - Sales growth vs market
   - Market share %
   - Price positioning

---

## 6. Export Data

From any page:

1. Click **Export** button
2. Choose **CSV** or **Excel**
3. File downloads immediately
4. Open in Excel/Google Sheets

## Common Tasks

| Task | Steps |
|------|-------|
| **Change date range** | Dashboard → Period dropdown |
| **Switch brands** | Dashboard → Brand dropdown (if multi-brand) |
| **Find top products** | Products → Sort by Revenue |
| **Check stock** | Products → View Stock Coverage column |
| **Export products** | Products → Export button |

## Tips

✅ **Use 90-day view**: Better for trend analysis
✅ **Compare to last year**: Accounts for seasonality
✅ **Export weekly**: Create offline reports
✅ **Monitor stock coverage**: Prevent stockouts
✅ **Focus on top 20%**: Use 80/20 rule
