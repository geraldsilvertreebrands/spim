# PIM Panel Quick Start Guide

**Get up and running in 5 minutes**

---

## 1. Login

- Go to your Silvertree platform URL
- Enter your email and password
- You'll be redirected to `/pim`

---

## 2. Create Your First Product

**Products** → **New Product**

Fill in:
- **SKU**: Unique code (e.g., `FTN-COC-500`)
- **Name**: Product name (e.g., `Organic Coconut Oil 500ml`)
- **Status**: Choose `Published` or `Draft`

Click **Create**.

---

## 3. Edit a Product

1. Click **Products** in sidebar
2. Search or browse for your product
3. Click the product row
4. Edit any fields
5. Click **Save**

---

## 4. Add Categories

**Categories** → **New Category**

- **Name**: Category name (e.g., `Oils & Fats`)
- **Parent**: Leave blank for top-level, or select a parent

Click **Create**.

Assign products to categories in the product edit form.

---

## 5. Sync with Magento

**Magento Sync** → **New Sync**

Choose:
- **Direction**: From Magento / To Magento / Bidirectional
- **What to sync**: Products, Categories, Attributes

Click **Start Sync**.

Monitor progress on the Magento Sync page.

---

## 6. Create an Attribute (Advanced)

**Attributes** → **New Attribute**

- **Name**: Machine name (e.g., `brand_name`)
- **Label**: Display name (e.g., `Brand Name`)
- **Type**: Text, Select, Number, etc.
- **Validation**: Add rules (required, unique, etc.)

Click **Create**.

---

## Common Tasks

| Task | Steps |
|------|-------|
| **Search products** | Products → Use search bar |
| **Delete product** | Edit product → Delete button |
| **Bulk edit** | Products → Side-by-Side Edit |
| **Run AI pipeline** | Pipelines → Select pipeline → Run Now |
| **View sync errors** | Magento Sync → Click failed sync run |

---

## Navigation

**Sidebar Menu**:
- **Products**: Manage product catalog
- **Categories**: Organize products
- **Attributes**: Define product fields
- **Attribute Sections**: Group attributes
- **Pipelines**: AI enrichment
- **Users**: Manage accounts (admin only)
- **Magento Sync**: Synchronize data
- **Review Queue**: Approve changes

---

## Tips

✅ **Use consistent SKU formats**: `BRAND-CATEGORY-NNN`
✅ **Fill required fields**: Reduces errors
✅ **Test Magento sync**: Start with 5-10 products
✅ **Use attribute sections**: Keep forms organized
